CaliTheme Wordpress Theme

What is this site for?

This is a wordpress theme for businesses to showcase themselves.

What does it do?

This is a customizable Wordpress theme. Each section is customizable through the Wordpress admin dashboard.

Features

  • Parallax scrolling for the header image
  • Modal popovers to describe portfolio images
  • Section to tell about your company
  • Section to showcase your team

Tech Used:

Flexbox

I used flexbox to responsively display the portfolio images.

JQuery

I used JQuery to handle the modal popovers for the portfolio images.

Bootstrap

I used Bootstrap to give my project a simple, responsive layout.
