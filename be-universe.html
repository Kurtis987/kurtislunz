BeUniverse Reverse Engineered WordPress Theme

What is this site for?

This is a reverse engineered WordPress theme. I reverse engineered it to sharpen my WordPress development skills.

What does it do?

This is a reverse engineered WordPress theme that provides information about the universe.

Features

  • Parallax scrolling for the gallery images
  • A responsive JavaScript navigation bar
  • Smooth scrolling for navigating smoothly to different sections of the page
  • Uses Flexbox and Grid for responsive sizing and positioning
  • A dummy form to subscribe

Tech Used:

Flexbox

I used flexbox to responsively display the images and text.

Grid

I used the grid display style to responsively position and style the earth and mars sections of the page

Bootstrap

I used Bootstrap to give my project a simple, responsive layout.

JavaScript

I used JavaScript to create a navigation bar that responds to scrolling and to enable smooth scrolling
