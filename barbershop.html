Barber Shop Reverse Engineered WordPress Theme

What is this site for?

This is a reverse engineered WordPress theme. I reverse engineered it to sharpen my WordPress development skills.

What does it do?

This is a reverse engineered WordPress theme that provides a web based store face for a barber shop.

Features

  • A responsive JavaScript navigation bar
  • Background images that change on hover for the navigation menu
  • Uses Flexbox for responsive sizing and positioning
  • A dummy form to book an appointment

Tech Used:

Flexbox

I used flexbox to responsively display the images and text.

JQuery

I used JQuery to change the background images on the navigation menu on hover.

Bootstrap

I used Bootstrap to give my project a simple, responsive layout.
