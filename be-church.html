BeChurch Reverse Engineered WordPress Theme

What is this site for?

This is a reverse engineered WordPress theme. I reverse engineered it to sharpen my WordPress development skills.

What does it do?

This is a reverse engineered WordPress theme that showcases a church.

Features

  • Parallax scrolling for some of the header images
  • A responsive JavaScript navigation bar
  • Uses Flexbox for responsive sizing and positioning
  • Bootstrap carousel for testimonials
  • A dummy form for the contact page

Tech Used:

Flexbox

I used flexbox to responsively display the images and text.

Bootstrap

I used Bootstrap to give my project a simple, responsive layout.

JavaScript

I used JavaScript to create a navigation bar that responds to scrolling
