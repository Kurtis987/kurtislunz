School Donations

What is this site for?

This site is for displaying school donation statistics in a dynamic fashion.

How does it work?

This site uses MongoDB to store data, and it uses D3,DC and Crossfilter to display graphs.

Tech Used

Some of the tech used includes:

MongoDB

I used MongoDB to store data documents.

D3,DC and Crossfilter

I used D3, DC and Crossfilter to display dynamic graphs with which the user can interact.
